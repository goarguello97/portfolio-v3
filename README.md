## Run Locally

Clone the project

```bash
  git clone https://github.com/goarguello97/portfolio-v3.git
```

Go to the project directory

```bash
  cd portfolio-v3
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
